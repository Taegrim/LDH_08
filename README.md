### 조작법
+ 이동 : WASD
+ 시점 회전 : 마우스
+ 점프 : 스페이스바
+ 달리기 : Shift
+ 일시정지 메뉴 토글 : Tab

### 게임 루프 및 아이템, 파티클, 사운드
+ `GameMode`, `PlayerController`, `Pawn`, `IMC`, `IA` 를 이용한 3D캐릭터
+ 아이템 인터페이스, `BaseItem`을 상속받아서 `Coin`, `HealingPotion`, `Mine` 아이템 구현
+ `SpawnVolume` 바운딩박스 내 랜덤한 위치에 누적합을 이용한 랜덤 아이템 생성
+ 아이템 데이터 테이블을 이용해서 레벨 별 다른 스폰 확률 적용
+ `Overlap` 이벤트를 이용한 충돌처리 및 다형성을 이용한 아이템 충돌 개별 처리
+ `GameState`를 이용한 레벨 처리, `GameInstance` 이용한 전역 데이터 관리
+ 아이템 충돌 시 파티클, 사운드 출력

### Wave
+ `GameState`에서 레벨마다 `Waves` 배열을 순서대로 처리함, 현재 3웨이브로 구성
+ 웨이브별로 소환하는 아이템 개수와 시간은 에디터에서 변경 가능함
+ 웨이브가 전부 끝나면 레벨이 전환되도록 변경함

### UI
+ 메뉴 버튼을 시작 버튼과 메인메뉴로 가는 버튼으로 분리
+ 시작 버튼은 `MainMenu`, `Pause`, `GameOver` 상태에 따라 다르게 동작하도록 함
+ `Tab`키를 눌러서 메뉴창으로 토글하도록 처리함

### 디버프 시스템
+ 디버프용 `SlowItem`, `VisionBlockItem` 아이템 추가
+ 플레이어는 디버프들이 저장되는 배열을 멤버로 가짐
+ 디버프가 생길 때 상태효과 갱신 타이머를 시작하고, 하나의 타이머로 모든 디버프 시간을 관리함
+ 슬로우, 화면 가리기 디버프가 있고 각각 중첩이 가능하며 별도의 시간을 가짐
+ 디버프별로 가장 긴 시간만 가져와서 화면의 우하단에 중첩개수와 남은시간을 표시함

### 동적 웨이브 환경
+ 웨이브 시작 시 웨이브 타입에 따라 환경 이벤트를 다르게 시작함
+ 현재 스파이크 함정과 폭격을 환경 이벤트로 추가함
+ 기존 `SpawnVolume`와 비슷하게 바운딩 박스를 이용해서 맵에 가시함정, 폭격을 생성
